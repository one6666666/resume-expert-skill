## 🆕 v2.0 → v3.0 更新内容（2026.06）

### v2.0（2026.06 基础版）
对照鱼皮《AI 时代满分的程序员简历是怎么样的》文章核心方法论：

#### AI 时代黄金法则
1. **求职意向带 AI 标签** — `Java 后端开发（具备 AI Agent 全栈开发经验）`
2. **作品集 + GitHub 必放头部** — 10 份简历 8 份没有，有就是区分度
3. **AI 技能 4 层结构** — AI 开发框架 → AI 应用实战 → AI 编程工具 → AI 编程模式
4. **项目必须上线** — 可访问地址是验证真实性的唯一方式
5. **每一条带量化数据** — QPS / DAU / 提升 X% / 成本降低 X%
6. **监控告警 + SEO/GEO** — "写完代码之后"的能力也是能力
7. **体现自主性** — 不只是参与，是主导、决策、交付

### v3.0（2026.06 实战增强版）✨ 新增 8 大洞察

#### 一、平台算法穿透
- **关键词精准匹配**：显性要求 100% 覆盖 + 行业前瞻性词汇补充，关键词密度 8%-12%
- **语义化表达升级**：`Python` → `Python + Flask + RESTful API 优化接口响应时间 40%`
- **格式机读友好**：禁用表格/图片/复杂排版，ATS 解析错误率 >30%

#### 二、HR 6秒黄金法则
- **页眉三要素**：姓名（18pt 加粗）+ 目标岗位（14pt 蓝色，与 JD 100% 一致）+ 即时联系方式
- **单屏呈现**：超过一屏内容被截断概率 38%，核心优势必须在第一屏
- **信息密度控制**：每行 ≤25 字，每段 3-4 行

#### 三、STAR + CAR 双框架
- **STAR**（完整项目）：情境 → 任务 → 行动 → 结果
- **CAR**（突破点）：挑战 → 行动 → 结果
- **量化公式**：「技术/行动 + 资产 + 指标 + 业务效果」

#### 四、平台差异化适配
| 平台 | 策略重点 |
|------|---------|
| BOSS直聘 | 在线简历关键词完整、每天 9-10 点刷新、定制化开场白 |
| 猎聘 | 突出中高端定位（负责人/架构/管理）、猎头沟通渠道 |
| 智联招聘 | 1 小时 300+ 份竞争，用 AI 工具 1 分钟完成优化 |

#### 五、稳定性与职业连续性
- 跳槽频率：5 年 3 跳为最佳，1 年 1 跳会标记为稳定性差
- 空窗期：超过 3 个月必须提供合理解释
- 职业曲线：呈现 3-5 年阶梯式成长，避免职位/薪资倒挂

#### 六、个人品牌与第三方背书
- GitHub：用"技术成长折线图"替代默认贡献格子
- 技术博客：展示"阅读-点赞-收藏-转化"漏斗数据
- 证书：遵循"岗位相关度 + 稀缺性"原则排序
- 社交证明：嵌入最高赞评论、官方转载记录、竞赛排名

#### 七、AI 辅助全链路优化
- **智能改写**：AI 简历姬 / Rezi — JD 对齐、匹配度评分、经历润色
- **多版本管理**：Teal / 超级简历 — "一岗一版"，动态调整内容重点
- **模拟面试**：AI 工具演练话术，确保简历内容面试时可自洽深挖

#### 八、跨场景精准适配
- **应届生**：课程设计商业化包装，用技术指标替代"课设"表述
- **转行者**：提炼可迁移技能（用户洞察/数据分析/项目管理）
- **管理岗**：OKR 与业务增长关联写法，技术动作映射业务 KR

## 📋 支持的六种模式

| 模式 | 触发条件 | 功能 |
|------|---------|------|
| **A — JD驱动优化** | 有简历 + 有JD | 解析JD（含AI相关检测+目标平台识别）→ 诊断（含AI时代+平台算法双检查）→ 逐条重写（含语义化升级） |
| **B — 简历审计** | 有简历 + 无JD | 30秒结论 + 分维度评分（含平台算法友好度+HR 6秒法则符合度）+ AI时代+平台双检查 + 逐项修改 |
| **C — JD反向生成** | 无简历 + 有JD | 收集信息（含目标平台）+ 反向定制生成（含平台差异化策略） |
| **D — 从零生成** | 无简历 + 无JD | 对话式信息收集（含目标平台询问）→ 含AI时代检查+平台策略生成 |
| **E — 面试准备** | 只要面试建议 | 高频考点 + 故事线（STAR展开）+ 弱点预判 + AI时代加分题 + 量化数据核查 |
| **F — 格式输出** | 只要格式转换 | HTML（含作品集/项目链接）+ DOCX + ATS纯文本（关键词密度 8%-12%） |
